# Bengali Literary Translation Style Guide

## Pronouns
- Use "তাঁরা" — NOT "তারা"
- Use "তাঁর" — NOT "তার"
- Use "তাঁদের" — NOT "তাদের"

## Verbs
- "বাস করেন" — NOT "থাকে"
- "বলে থাকেন" — NOT "বলে"
- "হয়ে উঠেছেন" — NOT "হয়েছেন" (for becoming something)
- "গড়ে তুলেছেন" — NOT "তৈরি করেছেন" (for building wealth)

## Vocabulary
- "স্বাবলম্বী" — NOT "স্বাধীন" (financially independent)
- "ঋণনির্ভর" — NOT "ক্রেডিটের প্রতি আসক্ত"
- "সম্মানজনক আবাসিক এলাকা" — NOT "চমৎকার এলাকা"
- "সুনামধন্য পাড়া" — NOT "ভালো এলাকা"
- "বহু মিলিয়ন ডলারের মালিক" — NOT "মাল্টিমিলিয়নেয়ার"
- "প্রথম প্রজন্মের ধনী" — NOT "প্রথম প্রজন্মের মিলিয়নেয়ার"
- "কর্মজীবন" — NOT "ক্যারিয়ার"
- "আর্থিক কৌশল" — NOT "ফাইন্যান্সিয়াল লিভারেজ"
- "ভোগবিলাস" — NOT "ভোগ"
- "ঋণের বোঝা" — NOT "ডেট"

## Idioms — NEVER translate literally
- "went cold turkey" → "একেবারে ছেড়ে দিলেন" / "হঠাৎ পুরোপুরি বাদ দিলেন"
- "saw the light" → "ভুল বুঝতে পারলেন" / "সত্য উপলব্ধি করলেন"
- "credit junkies" → "ঋণনির্ভর" / "ঋণের নেশায় আসক্ত"
- "workaholic" → "কাজপাগল" / "সারাক্ষণ শুধু কাজ নিয়েই মগ্ন"
- "show off success" → "সাফল্য প্রদর্শন" / "সাফল্য দেখানোর তাগিদ"
- "balance is their approach" → "ভারসাম্যই তাঁদের মূলমন্ত্র" / "ভারসাম্যপূর্ণ দৃষ্টিভঙ্গি"
- "financially independent" → "আর্থিকভাবে স্বাবলম্বী"

## Sentence Flow
- Merge short English sentences into longer, flowing Bengali sentences
- English short sentences → single compound Bengali sentence with connectors
- Avoid English word order (Subject-Verb-Object — follow Bengali S-O-V)
- Use "—" (dash) occasionally for dramatic pauses (common in Bengali nonfiction)
- Start paragraphs with "তবে", "কিন্তু", "আর", "আবার" where context fits

## Tone
- Maintain the author's original emotional tone
- If the English is assertive, keep it assertive in Bengali
- If reflective, keep it reflective
- Should read like Anisul Hoque or Muhammad Zafar Iqbal's nonfiction
- NOT like Google Translate
- NOT like news editorial Bengali
- Smooth, modern literary Bengali — not archaic

## What to Avoid
- Never use "আলোর দিকে এগিয়ে গেল" — use "ভুল বুঝতে পারলেন"
- Never use "ক্রেডিটের প্রতি আসক্ত" — use "ঋণনির্ভর"
- Never use "ফাইন্যান্সিয়াল লিভারেজ" — use "আর্থিক কৌশল" or rephrase
- Never use "ক্যারিয়ার" — use "কর্মজীবন" or "পেশাজীবন"
- Never keep English words unless no Bengali equivalent exists
- Never translate "millionaire mind" as "মিলিয়নেয়ার মাইন্ড" — use "মিলিয়নেয়ারের মানসিকতা" or "ধনীমানস"

## Overall Goal
The final text should make the reader forget it was ever written in English.
